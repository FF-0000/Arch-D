## What is Arch-D?

**Arch-D is simply Arch Linux — with Dinit instead of System-D.**

---

## The Rebellion 🔥

Arch-D exists as an **ACT Of Rebellion** ***AGAINST*** **System-D**

- **An init system should start processes**
- **UNIX Philosophy, MATTERS.** Do one thing and do it properly.
- **System-D Now manages your DNS, Logging, Login Sessions, Time-Sync, Device Nodes, and Bootloader.**
